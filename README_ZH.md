# AI超级内容工厂 - 前端演示

这是一个基于 Next.js 14、TypeScript、Tailwind CSS 和 shadcn/ui 构建的前端演示页面。

## 🎉 已完成功能

### ✅ 选题分析页面（首页）

包含完整的选题分析流程：

1. **搜索配置区域**
   - 关键词输入框
   - 关键词收藏功能（显示常用关键词）
   - 搜索逻辑选择（AND/OR）
   - 时间范围选择（1/3/7/15/30天）
   - 文章数量选择（10/20/50/100篇）
   - 筛选条件（阅读量、点赞数、指定公众号）

2. **分析进度展示**
   - 实时进度条
   - 当前分析状态
   - 已分析文章列表
   - 取消分析按钮

3. **洞察报告**
   - 点赞TOP5文章列表
   - 互动率TOP5文章列表
   - 高频词云TOP10（带频次）
   - 5个选题洞察推荐
     - 选题类型（热点型/干货型/情感型）
     - 建议标题
     - 内容方向
     - 目标受众
     - 写作角度
   - 一键创作按钮
   - 保存/导出功能

4. **历史报告**
   - 历史分析记录列表
   - 查看/导出/删除操作
   - 分页功能

### ✅ 导航栏

- Logo和应用名称
- 5个导航菜单：选题分析、内容管理、数据洞察、社媒账号、系统设置
- 用户头像下拉菜单（个人中心、退出登录）

## 🚀 如何访问

### 方式1：直接访问（服务器已启动）

项目已经在运行，直接在浏览器打开：

```
http://localhost:3000
```

### 方式2：重新启动服务器

如果服务器已停止，在项目目录运行：

```bash
cd content-factory
npm run dev
```

然后访问 http://localhost:3000

## 💡 交互演示

1. **开始分析**：点击"开始分析"按钮，会触发模拟的分析流程
   - 显示进度条（自动从0%到100%）
   - 显示当前分析状态
   - 完成后自动显示洞察报告

2. **收藏关键词**：点击收藏的关键词标签可以快速填充搜索框

3. **搜索逻辑切换**：可以在AND和OR之间切换搜索逻辑

4. **筛选条件**：勾选筛选条件复选框来启用相应的筛选

5. **一键创作**：点击选题卡片下方的"一键创作"按钮（将来会跳转到内容创作页面）

6. **历史记录操作**：可以查看、导出或删除历史报告

## 🎨 设计特点

- **现代化UI**：使用 shadcn/ui 组件库，简洁美观
- **响应式设计**：自适应不同屏幕尺寸
- **深色模式支持**：自动适配系统深色模式
- **流畅动画**：进度条、hover效果、过渡动画
- **清晰布局**：卡片式布局，信息层次分明

## 📁 项目结构

```
content-factory/
├── app/
│   ├── layout.tsx          # 根布局（包含导航栏）

│   └── globals.css          # 全局样式
├── components/
│   ├── ui/                  # shadcn/ui 组件
│   └── navbar.tsx           # 导航栏组件
└── lib/
    └── utils.ts             # 工具函数
```
